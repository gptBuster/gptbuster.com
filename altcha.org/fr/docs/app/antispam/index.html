The AntiSpam feature in the ALTCHA App primarily functions as a monitoring tool for ALTCHA’s security APIs—PoW Challenges and Spam Filter.

Key Features:

  • Analytics: Monitor and analyze request data.
  • Logs: View detailed logs of all processed requests.
  • Playground: Test and explore the Spam Filter API interactively.

AntiSpam helps you track and enhance your security setup for improved protection against spam and unwanted traffic.
